# 🚀 Maze Solver & Algorithm Visualizer

An interactive web application to visualize **pathfinding algorithms (BFS & DFS)** on a grid-based maze.

## 🎯 Features

* 🟢 Interactive Grid (20x20)
* ⬛ Draw Walls (Click & Drag)
* ⚡ Adjustable Animation Speed
* 🟡 BFS Traversal Visualization
* 🔵 DFS Traversal Visualization
* 🧠 Shortest Path Highlight (BFS)
* 📊 BFS vs DFS Time Comparison
* 🔄 Reset Grid

---

## 🧠 Algorithms Used

### 🔹 Breadth-First Search (BFS)

* Uses a Queue data structure
* Explores nodes level by level
* Guarantees the shortest path in an unweighted graph

### 🔹 Depth-First Search (DFS)

* Uses a Stack (or recursion)
* Explores as deep as possible before backtracking
* Faster in some cases but does not guarantee shortest path

## 🎮 How to Use

1. Click **Generate Grid**
2. Draw walls by clicking or dragging
3. Select BFS or DFS
4. Adjust speed using slider
5. Watch traversal and final path
